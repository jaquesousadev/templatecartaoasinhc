## How to use

This template utilizes Next.js and TailwindCSS and is written in TypeScript. In order to get developing, please run the following things in your terminal:

```bash
# change directory into the template
cd uranium-master

# install & run
yarn
yarn dev
# or
npm install
npm run dev
```
